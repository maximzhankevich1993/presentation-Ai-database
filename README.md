# База данных Презентатор ИИ

Схема базы данных, миграции и всё что связано с хранением данных для сервиса генерации презентаций с ИИ.

## Структура

| Таблица | Описание |
|:---|:---|
| `users` | Пользователи сервиса |
| `devices` | Устройства пользователей (защита от накрутки) |
| `presentations` | Созданные презентации |
| `payments` | Платежи и транзакции |
| `subscriptions` | Активные подписки |
| `referrals` | Реферальная программа |
| `gallery_likes` | Лайки в публичной галерее |
| `digests` | Отправленные дайджесты |
| `sessions` | Сессии пользователей |
| `audit_logs` | Логи безопасности |

## Установка

```bash
# Создать базу данных
createdb presentation_ai

# Применить схему
psql -d presentation_ai -f schema.sql

Безопасность

· Пароли хешируются через pgcrypto
· Токены сессий хранятся в виде хешей
· Логирование всех действий пользователей
· Защита от повторной установки через devices
· Блокировка после 5 неудачных попыток входа
